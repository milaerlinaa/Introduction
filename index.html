F4 Solving Quadratic Equations Introduction

Learning objectives

This unit focused on methods of solving quadratic equations. After completing Unit F4 you should

  • be able to solve quadratic equations that factorise into two linear terms
  • be able to use the formula for solving quadratic equations
  • understand how to complete the square for a quadratic and hence solve the equation.

Introductions

We can tell from Old Babylonian clay tablets dating from around 2000 BC that the Babylonians knew how to solve a pair of simultaneous equations of the form

$x + y = p, xy = q$

which is equivalent to the equation

$x^2 + q = p x$

In the Century BC, quadratic equations of the form $ax^2 = c$ and $ax^2 + bx = c$ were explored in ancient India, using geometric methods. Babylonian mathematicians from circa 400 BC and Chinese mathematicians from circa 200 BC used the method of completing the square (see Sections F4 .3) to solve quadratic equations with positive root, but did not have a general formula.

In 628 AD, Brahmagupta, an Indian mathematicians, gave the first explicit solutions of the quadratic equation but it was not until 1896 that the first reference to the general solution

$ax^2 + bx + c = 0$ ($a, b, c are constan, a \ne 0$)

as $x = {-b \pm \sqrt{b^2 - 4ac} \over 2a}$

was published by Henry Heaton in The American Mathematical Monthly (see information at

http://en.wikipedia.org/wiki/Quadratic_equation

and http://www.jstor.org/stable/2971099?seq=1 )

Key points and principles

  • There are three methods of solving quadratic equations :
  1. quadratic factorisation, where we can write

    2. $ax^2 + bx + c = a(x - p)(x - q)$ when $p$ and $q$ are rational numbers which has solution $x = p$ or $x = q$

  2. formula $x = {-b \pm \sqrt{b^2 - 4ac} \over 2a}$

  3. completing the square, where we write

    4. $ax^2 + bx + c = a \left(x + \frac{b} {2a} \right)^2 - \frac {b^2}{4a} + c$


  • Note that if $(x - a)(x - b) = 0$, then either $x - a = 0$ or $x - b = 0$, giving $x = a or x = b.$

  • Quadratic equations have 2, 1 or 0 real roots according to the value of "$b^2 - 4ac$":
  1. If $b^2 > 4ac$, there are 2 real distinct root

  2. If $b^2 = 4ac$, there is 1 (repeated) real root

  3. If $b^2 < 4ac$, there are no real root.

Facts to remember

  • If $(x - a)(x - b) = 0$ then $x = a$ or $x = b$

  • The formula for solving the quadratic equation

    • $ax^2 + bx + c = 0$

    is

    $x = {-b \pm \sqrt{b^2 - 4ac} \over 2a}$


  • The number of real roots of a quadratic is

    • 2 if $b^2 > 4ac$

    1 if $b^2 = 4ac$

    0 if $b^2 < 4ac$
